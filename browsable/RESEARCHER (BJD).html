RESEARCHER



This entity type represents a RESEARCHER.





Dependencies:

A RESEARCHER is an independent entity.

Attributes:
ResearcherUID - required - A unique identifier.
NamePointer - required - This is a link to the default NAME for this researcher.
Address - optional - a link to the primary LOCATION associated with this researcher.
Notes - optional

Relationships:
A RESEARCHER may have one or more NAME, LOCATION, or OBJECT entities.
