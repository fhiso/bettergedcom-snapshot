SOURCE

This entity type represents a SOURCE or citation of data.




Dependencies:
A SOURCE is an independent entity

Attributes:
SourceID - required - a unique identifier.
SourceTemplate - required - a template to use to layout the source parts.
SourcePart[xx] - required - An array of text fields giving the pieces of the the source citation.
Description - optional.

Relationships:
A SOURCE may be related to a PERSON, EVENT, GROUP, OBJECT, ROLE, NAME, LOCATION, or CHARACTERISTIC.
