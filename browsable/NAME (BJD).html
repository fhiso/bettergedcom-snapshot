NAME

This entity type represents a PERSON's NAME.




Dependencies:

An NAME belongs to one or PERSON.

Attributes:

NmeUID - required - a universally unique identifier.
NameTemplate - required - A classification and link to a template the name uses.
NameArray[xx]- optional - An array of up to 99 names. That should cover it.
Surety - optional - an indication of how sure the researcher is that this NAME truly belongs with the associated PERSON.
Description - optional - a description of the NAME.

Relationships:
A NAME must belong to 1 and only 1 PERSON.
