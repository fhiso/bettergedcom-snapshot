Please note that the information provided below is supplied by the individual member and should not be confused with the Member Pages created by Wikispaces when someone joins the Wiki.

  • Username - wiki user name. Embedded link to profile page if available. To create a profile page, use the member page template above. Your wikispaces profile is used as a default if you do not have a specific BetterGEDCOM one.

  • Country - country of residence.
  • TimeZone - simple UTC offset, and the local TimeZone mnemonic (in 'Country') which helps identify any Daylight Saving Time (DST).
  • Joined - Date joined wiki, expressed in ISO format. CST TimeZone of the US wiki server implied.

Each username links to your specific BetterGEDCOM profile page, if you have one, (e.g. http://www.wikispaces.com/user/view/XYZ) or your general wikispaces profilepage (e.g. XYZ).
