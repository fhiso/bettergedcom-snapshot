OBJECT

This entity type represents a text file, multimedia object, Internet link, or any number of other files which add substance to a PERSON, EVENT, LOCATION, or any other defined entity .




Dependencies:

An OBJECT belongs to one or more entities.

Attributes:

ObjectUID - required - a universally unique identifier.

ObjectClass - required - A classification of what kind of entity the object belongs to.(EVENT, RESEARCHER, OBJECTIVE, OBJECT, LOCATION, REPOSITORY, SOURCE., or CHARACTERISTIC). If an OBJECT needs to belong to more than one class, it will require multiple entries in the OBJECT list.
ObjectType - required - This tells us what the object is: a graphic, text file, movie, url, file, etc.
ObjectPointer - required - The location of the object.
Surety - optional - an indication of how sure the researcher is that this object truly belongs with the associated entity(/ies).
Description - optional - a description of the OBJECT.

Relationships:
A OBJECT must belong to 1 or more PERSON, LOCATION, EVENT, SOURCE, REPOSITORY, TASK(?), RESEARCHER, GROUP, or OBJECTIVE.
